# Gastos

Una aplicacion iOS que permite llevar el control de gastos diarios, escrita con SwiftUI

- Copia de seguridad en iCloud
- Recordatorios
- Reportes


![alt text](https://raw.githubusercontent.com/carlos-chaguendo/gastos-ios-app/main/imp.jpg)


# Requirements

- Xcode 12.4
- Swift 5.2
